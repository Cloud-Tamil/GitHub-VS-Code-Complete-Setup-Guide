# 🚀 GitHub & VS Code Complete Setup Guide

## 📖 Overview

This repository provides a complete beginner-friendly guide for connecting local projects with GitHub using Visual Studio Code and Git.

The main purpose of this project is to help developers, DevOps engineers, and cloud learners understand how Git and GitHub workflows operate in real-world development environments.

This guide covers:

- Installing Git and VS Code
- Configuring Git globally
- Connecting GitHub with VS Code
- Cloning repositories from GitHub
- Pushing local projects to GitHub
- Pulling latest changes from remote repositories
- Managing branches
- Solving common Git errors
- Understanding daily Git workflows
- Creating professional project structures
- Using `.gitignore` properly

This repository is designed especially for:

* Beginners learning Git & GitHub
* DevOps Engineers
* Cloud Engineers
* Software Developers
* Students and IT learners

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/username/repository-name.git
```

---

## Open Project

```bash
cd repository-name
code .
```

---

# 🔄 Git Workflow

## Add Files

```bash
git add .
```

## Commit Changes

```bash
git commit -m "Updated project"
```

## Push Changes

```bash
git push origin main
```
